en gros c'est un script qui automatisent certain trucs qui clc parce qu'on a la flm

# basicProgSetup.sh

ça installe Ubuntu et Git avec une clé SSH

# basicWebSetup.sh

ça installe PHP, Apache2, MySQL, et ça configure un utilisateur de base MySQL

# startWeb.sh

ça lance juste WSL et les services Apache2 et MySQL en demandant le mot de passe

#startOther.sh

ça ouvre WSL dans le dossier racine de ou vous programmez, faut modifier le chemin d'accès par contre sinon ça marche pas
